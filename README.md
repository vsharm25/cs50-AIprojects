# CS50 AI Projects

A collection of AI projects completed as part of Harvard's CS50 Introduction to Artificial Intelligence with Python.

---

## Project 0 — Degrees of Separation

Finds the shortest path between two actors using Breadth-First Search (BFS), based on the Six Degrees of Kevin Bacon concept.

### Concepts used
- Breadth-First Search (BFS)
- Graph traversal
- State space search

### How to run
Run the following command in your terminal:

    python degrees.py large

### Example output

    3 degrees of separation.
    1: Emma Watson and Tracey Ullman starred in The Tale of Despereaux
    2: Tracey Ullman and Michelle Pfeiffer starred in I Could Never Be Your Woman
    3: Michelle Pfeiffer and Jennifer Lawrence starred in Mother!
